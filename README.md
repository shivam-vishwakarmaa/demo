# Premium Todo App 🚀

A beautiful, functional, and responsive Todo application built with pure Vanilla HTML, CSS, and JavaScript.

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Version: 1.0.0](https://img.shields.io/badge/Version-1.0.0-blue.svg)

## ✨ Features

-   **Modern UI/UX**: Sleek dark mode design with glassmorphism effects and smooth animations.
-   **Dynamic Interactions**: Real-time task adding, toggling, and deletion with visual feedback.
-   **Filtering**: Easily view all tasks, only active ones, or completed ones.
-   **Persistence**: Uses `localStorage` to save your tasks locally in the browser.
-   **Responsive**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom properties, Flexbox, Grid, and Keyframe animations.
-   **JavaScript (ES6+)**: Functional programming for DOM manipulation and local storage.
-   **Fonts**: Outfit (Google Fonts).

## 🚀 Getting Started

No installation or build process is required. You can run this project locally in seconds.

### Prerequisites

-   A modern web browser (Chrome, Firefox, Safari, Edge, etc.).

### Running the Project

1.  **Clone the Repository** (if you haven't already):
    ```bash
    git clone https://github.com/shivam-vishwakarmaa/demo.git
    ```
2.  **Navigate to the project folder**:
    ```bash
    cd demo
    ```
3.  **Open the App**:
    Simply double-click the `index.html` file or right-click and select "Open with your browser."

    Alternatively, if you have VS Code, you can use the **Live Server** extension to host it locally.
